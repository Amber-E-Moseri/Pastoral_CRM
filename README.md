# Pastoral_CRM
A lightweight CRM system for tracking calls, follow-ups, and engagement workflows.

## 🚀 Features
- Smart call logging system
- Automated follow-up tracking
- Daily & weekly email reminders
- Priority-based task sorting
- Dashboard with real-time insights

## 🧠 Problem it solves
Managing outreach manually leads to missed follow-ups and poor tracking.
This system ensures consistent engagement and accountability.

## ⚙️ Tech Stack
- Google Apps Script
- HTML/CSS (frontend UI)
- Google Sheets (database)
- Gmail API (notifications)
